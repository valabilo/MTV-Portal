## Testing Data

To test the system, you can manually add a row to Quiz_Attempts:

```
test-attempt-001	2026-05-03T10:00:00.000Z	John Doe	john@example.com	4	5	80%	PASSED		2026-05-03T10:00:00.000Z
```

Then run the `checkForNewCertificates()` function in Apps Script to test certificate generation.
